Pockebot!
=========


Simple bot that save articles direct from your favorite
messanging app to Pocket. Additional this bot has some
extra features.


What I need?
------------

* python3
* mongodb
* pymongo
* [pocket](https://github.com/tapanpandita/pocket)
* [telepot](https://github.com/nickoala/telepot)
